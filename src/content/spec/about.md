# About

안녕하세요, 로봇 소프트웨어 엔지니어 **정현우**입니다. 👋

자율주행 로봇의 **SLAM과 Localization**을 개발하고 있습니다.
현장에서 마주친 문제와 공부한 내용을 이 블로그에 기록합니다.

## 하는 일

- 2D/3D LiDAR SLAM 개발 및 현장 적용
- Localization (AMCL, ICP, EKF 기반 상태 추정)
- 물류 이송 로봇 · 자율 피킹 로봇의 자율주행 스택
- LiDAR · IMU · 초음파 등 센서 검토와 캘리브레이션

## 주요 프로젝트

- **SLAM-Toolbox 기반 Lifelong SLAM** — 장기 운영 환경에서의 지도 갱신
- **Laser Filter & Merger** — 다중 2D LiDAR 필터링·병합 파이프라인
- **Encoder Dead Reckoning** — 휠 오도메트리 기반 추측 항법
- **Multi-Robot Map Merge** — 다중 로봇 지도 병합

## 요즘 관심사

- ROS2 기반 SLAM 스택 (SLAM Toolbox, Nav2)
- LiDAR-Inertial Odometry (LIO-SAM, GLIM)
- 학습 기반 SLAM과 VLA 모델의 로보틱스 적용
